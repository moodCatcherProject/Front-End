# ✨무드캐쳐 (mood catcher)

- **무드 캐쳐를 꿈꾸는 모든 일반인들을 위한 커뮤니티 사이트.**
- 패션 커뮤니티, 옷의 후기 및 리뷰 취합.

# 📆 프로젝트 기간

- 2022/08/26 ~ 2022/10/07

# 1. 팀 소개

- React, Node.js 기반
- 개발인원: 7명
  1. Front-end: 박준수, 신수정
  2. Back-end: 조권영, 황수민, 이수범
  3. Designer: 김유나
  4. PM: 김승빈

# 2. 프로젝트 설명

- 개발언어 : JavaScript, React
- 배포환경 : S3, Route 53 ,CloudFront
- 협업 툴 : git / notion / figma
